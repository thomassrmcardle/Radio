**Welcome to Radio!**
----
This is a simple project designed to be a client-side, TTS based radio application to be used alongside Spotify in order to create a personalized experience for all users.

The system is free to use, so if you fancy having a look, feel free and I hope you enjoy!

**Requirements**
---
- Windows OS
- Spotify Premium (unfortunately this is a limitation due to the APIs and there isn't a known work around)
